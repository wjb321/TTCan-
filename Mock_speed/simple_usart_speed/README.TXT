
此处只是简易的通过设置不同的数据，然后将数据通过
usart的形式发送到ttcan板子上面去，之前通过定时器构造出所谓的
T/M等方法，但是由于数据变化的范围很小，比如现在车速是5 m/s, 识别车是减速的范围应该是
当车速为Vt 为： 4.85m/s < Vt < 5m/s, 即速度变化 deltaVt = 0.15 m/s. 这个值转化成转速满足公式：
Vt = 2 * pi * r * n (转速) ->  17.153 < vt < 17.684  -> delta n = 0.53 m/s. 但是如果通过定时器
中更改脉冲长度，然后通过转速公式来计算时候，会因为数据的变化范围是在1及以上的层级变动的，所以不能很好
显示这种速度的变化程度。所以应该最好采用真是的编码器轮子转动来进行相应的数据分析。因此此处单一的采用
按键加减然后将数据发送到串口的形式来进行。
